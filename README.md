# Real-Time File Monitor & Alert System

## Features
- Monitors file create/modify/delete events
- Logs events with timestamps
- Detects suspicious deletion patterns
- Stores audit logs
- Runs continuously in background

## Tech Used
- Python
- Watchdog
- Logging module

## How to Run
1. Activate venv
2. Run: python monitor.py
3. Use another terminal to create/modify/delete files
